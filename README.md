### :dart: Features
- Real-Time Messaging with Socket.io
- Group Chats and Chat Rooms
- User Authentication and Profiles
- MongoDB Atlas Integration
- Responsive Design with Chakra UI
- Cloud-Based Image Storage with Cloudinary
